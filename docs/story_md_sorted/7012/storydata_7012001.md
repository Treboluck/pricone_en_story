# フォローガールアユミ
魔物をなぎ倒しながら順調に進んでゆくエリコたち。{player}がハンカチを落とす出来事などがあったが、後ろからついてきているアユミのフォローによって知らぬうちに解決しているのだった。
  
## Subtitle: フォローガールアユミ
  
**ナレーション：**  
ルナの塔を進むエリコと{player}。  
その前には無数の魔物が立ち塞がっていた。  
  
**魔物：**  
──グオオオオオッ！  
  
**エリコ：**  
邪魔ですわ。  
  
**魔物：**  
──グオオオオオッ！

その程度ですの？  
  
**魔物：**  
──グオオオオオッ！  
  
**エリコ：**

**魔物：**  
──ギャオオオオオッ！  
  
**エリコ：**  
クスクス。  
もうおしまいですか？  
動かないなら、私の方から行っても──  
  
**魔物：**  
…………  
  
**エリコ：**  
あら？  
残りの魔物が逃げ出してしまいましたわね。  
  
**エリコ：**  
私とあなた様の逢瀬を邪魔することがいかに罪深いか、  
理解していただけたようですね。  
  
**エリコ：**  
これで私とあなた様……  
二人だけの世界ですわ……クスクス……  
  
**Choice: (1)**  顔が汚れてるよ。  
`Tag 1`  
**エリコ：**  
戦っているときに汚れてしまったのでしょう。  
恥ずかしいところをお見せしてしまいました。  
  
**エリコ：**  
……そうですわ。  
もしよろしければ、あなた様が直々に……拭いていただけませんか？  
  
**エリコ：**  
あなた様が自ら、私についた汚れをきれいにしてくださる……  
こんなにも喜ばしいことはありません……  
  
**エリコ：**  
もちろんいいよ、ですか。  
クスクス……そう言ってくださると思っていました。  
では、よろしくお願いいたします。  
  
**エリコ：**  
……？  
どうかされましたか？  
何かお探しのようですが……  
  
**Choice: (2)**  持っていたハンカチがない。  
`Tag 2`  
**エリコ：**  
な、なんてことでしょう……！  
どこかで落としてしまわれたと……？  
  
**エリコ：**  
では私のハンカチで……  
いえ、  
あなた様の身に着けていた私物で私を清めてこそ……！  
  
**エリコ：**  
すぐに探しに……  
あら？  
あなた様、そちら側のポケットに入っているのは、  
そのハンカチではありませんか？  
  
**エリコ：**  
クスクス。  
どうやらうっかりしていたようですね。  
小首をかしげるあなた様もかわいらしいですわ……  
  
**エリコ：**  
では、改めて……お願いいたします。  
あなた様……  
  
**エリコ：**  
力強くも優しい手つき……

身も、心までも、きれいになった気分ですわ……

──グオオオオオッ！  
  
**エリコ：**  
…………  
  
**エリコ：**  
私は今、

引っ込んでいて……くださいませんこと……？  
  
**魔物：**  
…………  
  
**エリコ：**  
さぁあなた様。  
引き続き参りましょう。  
この先も楽しくなりそうですわね……♪  
  
**ナレーション：**  
身も心もきれいになったエリコは、  
ご機嫌そうに先へと進むのであった。  
そして、その後をついてゆくのは──  
  
**アユミ：**  
先輩メモ。  
エリコさんが魔物と戦っているとき、先輩がハンカチを落としたので、  
拾って先輩のポケットに戻しておいた。  
  
**アユミ：**  
戦闘の邪魔にならないようにこっそりと行動したけど、  
見つからずに済んだみたい。  
  
**アユミ：**  
……先輩のハンカチはきれいに畳まれていて、  
すごくいい匂いがした。  
  
**アユミ：**  
って何書いてるんだろう私……！  
ちちっ、違うんですよ先輩っ、別に私、ハンカチを嗅いだわけじゃ……！  
  
**アユミ：**  
それにしてもエリコさん、すごかったなぁ～……  
魔物をあっという間に倒しちゃって、  
しかも最後なんて言葉だけで魔物をなんとかしちゃうし……！  
  
**アユミ：**  
私なんて、  
魔物と向かい合っただけで体がブルブル震えちゃって……  
  
**アユミ：**  
でも、あんなに強いエリコさんもかわいいところがあるみたい。  
魔物と戦い終わった後、汚れた手で顔をぬぐって、  
逆に顔を汚しちゃったりして。  
  
**アユミ：**  
……でもなんか、手が汚れてるのを確認した後、  
自分から顔を触ってたような？  
ううん、気のせいですよね！  
  
**アユミ：**  
私もあんな風に先輩に、顔を近づけて……  
  
**アユミ：**  
ひゃ、ひゃぅっ……！  
か、考えるだけで頬が熱く……！  
  
**アユミ：**  
そっ、そろそろ進まなきゃ！  
先輩との話題作りのために、頑張らないと！
