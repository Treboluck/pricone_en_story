# できる大人の携帯食
順調に探索が進みしばし休憩する一行。ユカリが軽食をふるまうが、その中身はおつまみのようなものばかりだった。その様子にミサキは、ユカリが本当に自分の理想の大人のレディなのか疑問を抱き始める。
