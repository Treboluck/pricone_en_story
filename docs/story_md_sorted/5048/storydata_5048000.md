# 幸せのエンジェルは突然に
ゲーム好きの集まるオフ会に参加することになったヨリだが、上手く喋れるか不安を感じていた。そんな彼女の前にエンジェルのラフィが現れる。ラフィはヨリの願いを叶えると言っていきなり矢を放つのだった。
